# 📖 Overview

MealMate AI is an AI-powered meal planning application that helps users create personalized daily meal plans based on their dietary preferences, cooking time, available ingredients, budget, and nutrition goals.

Instead of spending time deciding what to cook, users receive a complete AI-generated daily meal plan in seconds.

The application generates:

- 🍳 Breakfast
- 🥗 Lunch
- 🍛 Dinner
- 🍎 Healthy Snack
- 🛒 Grocery List
- 🔄 Ingredient Substitutions
- 💰 Budget Analysis
- 🥦 Nutrition Summary

---

# ❗ Problem Statement

Many people struggle to decide what to cook every day.

They often face challenges such as:

- Limited cooking time
- Budget constraints
- Lack of meal planning
- Missing ingredients
- Dietary restrictions
- Nutritional imbalance

Traditional meal planning is repetitive and time-consuming.

MealMate AI solves this problem using Google's AI models to generate personalized meal recommendations instantly.

---

# 💡 Solution

MealMate AI transforms user preferences into a structured daily meal plan.

Users simply enter:

- Diet Preference
- Food Goal
- Budget
- Cooking Time
- Cuisine
- Available Ingredients
- Allergies

The AI analyzes these preferences and generates a complete personalized meal plan.

Install dependencies
